# ЦТТ «Импульс» — сайт теннисного клуба

Премиальный сайт теннисного клуба «Импульс» (Мытищи, рядом с национальным парком
«Лосиный Остров»). Современный минимализм, тёмно-зелёный + лайм, mobile-first.

## Стек

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS** + UI-компоненты в стиле shadcn/ui
- **Framer Motion** — плавные анимации
- **lucide-react** — иконки

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # продакшн-сборка
npm run start    # запуск собранной версии
```

## Дизайн-система

Цвета и токены заданы в `tailwind.config.ts` и `src/app/globals.css`:

- `forest.900` `#0F3D2E` — основной тёмно-зелёный
- `lime.DEFAULT` `#B8E03E` — акцентный лайм
- `ink` `#111827` — нейтральный тёмный текст
- Шрифты: **Sora** (заголовки, `font-display`) + **Inter** (текст, `font-sans`)

Базовые компоненты: `src/components/ui/` (`button`, `card`, `badge`).

## Структура

```
src/
  app/                  # layout, globals.css, главная страница
  components/
    ui/                 # дизайн-система (кнопки, карточки, бейджи)
    media/              # MediaImage, MediaVideo (гибкие медиа-компоненты)
    hero/               # HeroMedia (переключение фото/видео)
    sections/           # секции страницы (hero, services, booking, ...)
  lib/
    media.ts            # ⭐ ЕДИНЫЙ КОНФИГ всех путей к фото/видео
    site.ts             # навигация и контактные данные
    utils.ts            # cn()
public/
  images/{hero,services,team,facilities,gallery}/
  videos/
media-raw/              # исходники для обработки (не публикуются)
```

## Работа с медиа (важно)

Все изображения и видео подключаются **через компоненты с пропсами** и берут пути
из одного места — `src/lib/media.ts`. Чтобы заменить плейсхолдер на реальное фото:

1. Положите файл в нужную папку `public/images/<раздел>/`.
2. Обновите путь в `src/lib/media.ts`.

Менять сами секции/вёрстку не нужно — компоненты подстраиваются под любые пропорции
за счёт пропсов `ratio` и `fit` у `MediaImage`.

### Hero: фото ↔ видео

В `media.ts` у `media.hero` есть поле `kind`:

- `kind: "image"` — фон-фото;
- `kind: "video"` — фон-видео (mp4/webm) + постер.

Компонент `HeroMedia` сам выберет нужный режим — переключение в одну строку.

> Все изображения — **реальные фото клуба**, обработанные в едином премиальном
> стиле (тёмно-зелёный грейдинг + лайм-акценты) и экспортированные в `.webp`.
> Исходники и пайплайн обработки — в `media-raw/` (скрипты `_download` / `_process`,
> см. `media-raw/README.md`). Сгенерированных плейсхолдеров на сайте не осталось.
