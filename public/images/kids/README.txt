Детские тренировки по теннису
==============================

На сайте используются WebP (оптимально для скорости):

  kids-training-1.webp
  kids-training-2.webp

При замене фото:
  1. Положите новый .jpg в эту папку
  2. Запустите: npm run optimize:images
  3. Пути — в src/lib/media.ts → media.kids

Рекомендуемые настройки конвертации: quality 80–85, max width 1920px.
